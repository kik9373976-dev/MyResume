# Frontend Resume

Интерактивное резюме Русалины Музафаровой на React + Vite. В проекте есть адаптивная навигация, страницы резюме, работ, профиля и контактов, а также кинематографичные route- и scroll-переходы.

## Запуск

```bash
npm install
npm run dev
```

Для production-сборки:

```bash
npm run build
npm run preview
```

## Как добавить проект

1. При необходимости добавьте превью в `src/assets/projects`.
2. Импортируйте изображение в `src/data/projects.js`.
3. Добавьте новый объект в массив `projects`: укажите уникальный `id`, название, описание, технологии и ссылки.
4. Установите `featured: true`, если проект должен появиться на главной.

Контактные данные находятся в компонентах футера и страницы контактов, а ссылка на GitHub — также в данных проектов.

Фото для резюме подключено из `pictures/myPhoto.PNG`. Тексты опыта находятся в `src/pages/Home.jsx` и намеренно не содержат выдуманных компаний или должностей — их можно заменить реальными данными.
